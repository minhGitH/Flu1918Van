# Flu1918Van

Supporting code for the undergraduate thesis manuscript: 
**Mathematical Modelling of the 1918 Flu Pandemic in Vancouver, British Columbia,  
Le Tuan Minh Nguyen, Simon Fraser University**,

Supervised by **Jessica Stockdale, Simon Fraser University**.


## Content

Code to generate each figure in the manuscript, along with weekly influenza mortality data and daily mean temperature data in Vancouver, British Columbia, Canada, during the 1918 pandemic. Weekly influenza mortality data were digitized from the Medical Health Officerâ€™s report from 1914 to 1919 in the Vancouver Archives by **Jessica Stockdale**. And, daily mean temperature data were extracted from the daily weather data recorded by the Vancouver PMO weather station in British Columbia in 1918 and 1919 [[http://www.bccdc.ca/health-info/diseases-conditions/covid-19/data](https://climate.weather.gc.ca/climate_data/daily_data_e.html?hlyRange=%7C&dlyRange=1898-10-01%7C1979-05-31&mlyRange=1898-01-01%7C1979-12-01&StationID=898&Prov=BC&urlExtension=_e.html&searchType=stnName&optLimit=yearRange&StartYear=1918&EndYear=1919&selRowPerPage=25&Line=0&searchMethod=contains&txtStationName=vancouver&timeframe=2&Day=29&Year=1919&Month=1#)](Government of Canada).
